# Rock Paper Scissors ✊✋✌️
A simple Rock-Paper-Scissors game built with vanilla HTML, CSS, and JavaScript. Play against the CPU, track scores, and view match history.

**Live Demo:** git clone https://github.com/yourusername/rock-paper-scissors.git 

### Features 
1. Play vs CPU- CPU chooses randomly using `Math.random()`
2. Score Tracking - Real-time player and CPU scores
3. Game History - Latest moves shown first with emoji icons
4. Reset Function - Clear scores and history to start fresh
5. Responsive UI - Works on desktop and mobile

### Tech Stack
-  HTML5 - Game structure 
- CSS3 - Styling and layout
- Vanilla JavaScript - Game logic, DOM manipulation
